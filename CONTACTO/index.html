Explorando nuevos horizontes literarios

Thlantis nace como un proyecto transmedia diseñado para incentivar el interés local por la lectura, la escritura y el análisis profundo de obras clásicas y géneros contemporáneos.

Conectamos diferentes formatos digitales para ofrecerte una experiencia literaria completa, desde artículos web minuciosos hasta contenido audiovisual dinámico.
